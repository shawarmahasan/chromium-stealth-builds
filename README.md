## 📦 About chromium-stealth-builds

chromium-stealth-builds offers pre-built Chromium browser versions with special fingerprint patches. These patches make the browser harder to detect by websites that try to block automated or unusual users. This makes it useful for web scraping, browser automation, and testing tools.

This project provides ready-to-use builds for Linux x64 and macOS arm64 systems. It is a component used by cloakbrowser, which downloads these builds automatically when it runs. This repository lets you get the same stealth Chromium versions without extra setup.
